Maven Plugin

Goals available for this plugin:

2 goals
Goal Description
dionysus-example:help Display help information on dionysus-example-maven-plugin.
Call mvn dionysus-example:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
dionysus-example:helper Helper mojo.

Usage

You should specify the version in your project’s plugin configuration:

<project>
  <build>
    <!-- plugin management -->
    <pluginMangement>
      <plugins>
        <plugin>
          <groupId>org.sonatype.dionysus</groupId>
          <artifactId>dionysus-example-maven-plugin</artifactId>
          <version>1-SNAPSHOT</version>
        </plugin>
      </plugins>
    </pluginMangement>

    <!-- plugin execution -->
    <plugins>
      <plugin>
        <groupId>org.sonatype.dionysus</groupId>
        <artifactId>dionysus-example-maven-plugin</artifactId>
        <version>1-SNAPSHOT</version>
      </plugin>
    </plugins>
  </build>
</project>
